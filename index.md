# Privacy Policy
Eyemaster stores the minimum data required to provide moderation functionality.

## Information Stored
We collect limited Discord-related data strictly for moderation purposes:
- **User IDs**: Unique numerical Discord account identifiers belonging to individuals who receive an infraction (i.e. warnings) to maintain accurate moderation histories.
- **Staff IDs**: Unique numerical Discord account identifiers belonging to Discord server staff members to track who issued a specific infraction.
- **Server IDs**: Unique numerical Discord server identifiers to track which infractions belong to a specific server.
- **Moderation records**:
    - Reason provided by staff
    - Expiration date (if applicable)
    - Timestamp metadata (creation and update dates)
    - Record version metadata
- **Server configuration data**:
    - Moderation log channel ID
    - Moderator role ID

Eyemaster does not collect message contents, direct messages, attachments, or unrelated Discord content beyond the moderation and configuration data described above.

Certain commands may temporarily access publicly available Discord account or server information to function (for example, commands that fetch information about a user). This information is not stored.

## Purpose of Data Collection
This data is stored solely to:
- Track moderation actions within a server.
- Maintain a history of infractions on a per-user basis.
- Allow server staff to review and enforce moderation decisions consistently.

## Data Sharing
No data is sold, shared, or provided to third parties.

## Data Retention
Moderation records are retained as long as necessary for server moderation and rule enforcement. Because this bot operates as a server moderation tool, certain records may be preserved indefinitely to maintain moderation history and prevent abuse.

Server staff may remove or update specific records where needed.

## Data Access and Deletion Requests
Users or server administrators may request information about stored data or request removal of specific records by contacting the bot developer through the provided contact method.

Some moderation records may not be fully deletable if they are required to maintain server moderation integrity.

## Support
For questions, data requests, or concerns, you may contact the the developer via [Support](https://eyemaster.github.io/Support).
